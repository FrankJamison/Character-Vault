About

A small full-stack demo app for auth + CRUD using vanilla JS and Express.

What you can do

  • Register, sign in, and sign out
  • Create and delete characters
  • Create and delete tasks
  • Update account settings

Tech stack

  • Frontend: HTML + CSS + vanilla JS (Bootstrap 4)
  • Backend: Node.js + Express
  • Auth: JWT stored in localStorage
  • Persistence: JSON file database
